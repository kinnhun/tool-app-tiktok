📖 Hướng dẫn sử dụng

Các bước sử dụng tool từ A đến Z

1️⃣ Tạo Google Sheet

Tạo file Google Sheet với cấu trúc cột theo mẫu:
A - STT | B - Link TikTok | C - Trạng thái | D - Tên sản phẩm | E - Giá hiện tại | F - Giá gốc | G - Giá sale | H - Link sản phẩm | I - Tên shop | J - Cập nhật lúc | K - Ghi chú

Nhập link TikTok vào cột B, để trạng thái cột C là "Chưa xử lý".

2️⃣ Share quyền cho tool

Copy email Service Account ở thanh bên trái.
Mở Google Sheet → Bấm nút Share → Thêm email đó với quyền Editor.
Đây là bước bắt buộc để tool có thể đọc và ghi dữ liệu vào sheet.

3️⃣ Kết nối Sheet vào tool

Vào mục Kết nối Sheet → Dán link Google Sheet → Kiểm tra kết nối → Chọn tab → Chọn cột → Lưu cấu hình.

4️⃣ Chạy lấy giá

Vào Tổng quan hoặc Quản lý cấu hình → Bấm nút ▶ Chạy.
Tool sẽ tự đọc link, mở video TikTok, tìm sản phẩm, lấy giá và ghi vào sheet.
Các link đã thành công sẽ được bỏ qua ở lần chạy sau. Muốn lấy lại giá, đổi trạng thái thành "Cần cập nhật lại".

5️⃣ Sửa lỗi "Storage Quota Exceeded" (Dành cho Gmail cá nhân)

Nếu bạn dùng Gmail cá nhân, Service Account sẽ không có dung lượng để lưu ảnh. Hãy làm theo các bước sau:

  1. Truy cập script.google.com và chọn Dự án mới.
  2. Dán đoạn mã bên dưới vào trình biên tập (Xóa hết mã cũ):
function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var folder = DriveApp.getFolderById(data.folderId);
  var blob = Utilities.newBlob(Utilities.base64Decode(data.base64), 'image/jpeg', data.filename);
  var file = folder.createFile(blob);
  file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
  return ContentService.createTextOutput(JSON.stringify({
    success: true, 
    url: "https://drive.google.com/uc?export=view&id=" + file.getId()
  })).setMimeType(ContentService.MimeType.JSON);
}
  1. Bấm Deploy (Triển khai) -> New Deployment.
  2. Chọn loại Web App. Cấu hình:
    • Execute as: Me
    • Who has access: Anyone
  3. Copy Web App URL và dán vào phần Cài đặt của tool.
